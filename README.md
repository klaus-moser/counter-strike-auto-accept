# csgo-ahk

##### Requirements
- Requires AutoHotkey v2.0
- https://www.autohotkey.com/download/ahk-v2.exe

##### Installation
- Click and install AutoHotKey via the `ahk-v2.exe`
- Double-Click the `CS2-Clicker.ahk` to load script


## Features
- START / STOP via [Strg] + [F11]
- ✨Magic ✨
